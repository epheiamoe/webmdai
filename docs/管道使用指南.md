# WebMDAI 管道（Pipe）使用指南

> 管道功能允许你通过标准输入（stdin）和输出（stdout）与其他命令组合使用，实现更灵活的处理流程。

---

## 什么是管道？

管道是Unix/Linux哲学的核心概念：
> "每个程序只做好一件事，然后通过管道组合起来"

**对比**：

| 方式 | 适用场景 | 示例 |
|-----|---------|------|
| **工作流** | 重复性任务，需要保存配置 | 翻译整部轻小说 |
| **管道** | 临时性处理，快速组合 | 处理几个链接，测试效果 |

## 支持的管道命令

### 1. `fetch pipe` - 从管道读取URL

**基本用法**：
```bash
# 从echo传递单个URL
echo "https://example.com" | webmdai fetch pipe -n mytask

# 从文件读取多个URL
cat urls.txt | webmdai fetch pipe -n mytask -st
```

**参数**：
- `-n, --name`: 任务名称（必填）
- `-r, --reader`: 爬取服务商（默认: jina）
- `-s, --separate`: 每个网页保存为单独文件
- `-t, --together`: 所有网页合并为一个文件
- `-st`: 同时生成单独文件和合并文件

**示例 - 配合其他命令**：
```bash
# 从剪贴板获取URL（Windows）
Get-Clipboard | webmdai fetch pipe -n clipboard_task

# 从grep过滤的URL爬取
grep "https://" bookmarks.html | webmdai fetch pipe -n bookmarks -st
```

---

### 2. `llm pipe` - 从管道读取内容处理

**基本用法**：
```bash
# 直接翻译文本
echo "Hello World" | webmdai llm pipe -t translate

# 翻译文件内容
cat article.md | webmdai llm pipe -t translate -o translated.md

# 生成摘要
cat long_article.md | webmdai llm pipe -t summarize > summary.md
```

**参数**：
- `-m, --model`: 模型别名（默认使用默认模型）
- `-t, --task`: 任务类型（translate/summarize/explain/abstract）
- `-p, --prompt`: 自定义提示词
- `-o, --output`: 输出文件（不指定则输出到stdout）

**示例 - 配合其他命令**：
```bash
# 爬取后立即翻译
echo "https://example.com" | webmdai fetch pipe -n temp -t | \
  cat temp/*合并.md | webmdai llm pipe -t translate -o result.md

# 批量处理并合并
cat urls.txt | webmdai fetch pipe -n batch -t | \
  cat batch/*合并.md | webmdai llm pipe -t summarize > summary.md
```

---

### 3. `deal pipe` - 从管道读取内容替换

**基本用法**：
```bash
# 替换文本
echo "hello world" | webmdai deal pipe -f "world" -r "Python"

# 删除广告文字
cat article.md | webmdai deal pipe -f "广告" -o cleaned.md

# 正则替换
echo "123abc456" | webmdai deal pipe --re -f "\d+" -r "[数字]"
```

**参数**：
- `--text`: 普通文本模式（默认）
- `--re`: 正则表达式模式
- `-f, --find`: 查找内容（必填）
- `-r, --replace`: 替换内容（留空表示删除）
- `-o, --output`: 输出文件（不指定则输出到stdout）

**示例 - 配合其他命令**：
```bash
# 清理爬取内容
cat article.md | webmdai deal pipe -f " unwanted text" -o cleaned.md

# 批量替换
cat *.md | webmdai deal pipe --re -f "\\s+" -r " " > merged.txt
```

## 实际使用案例

### 案例1：快速翻译单个网页

```bash
# 一步完成：爬取 + 翻译
echo "https://example.com/article" | webmdai fetch pipe -n temp -t && \
cat temp/*合并.md | webmdai llm pipe -t translate -o translated.md && \
rm -rf temp/

# Windows PowerShell 版本
Write-Output "https://example.com/article" | webmdai fetch pipe -n temp -t
cat temp/*合并.md | webmdai llm pipe -t translate -o translated.md
Remove-Item -Recurse -Force temp
```

### 案例2：批量处理URL列表

假设你有 `urls.txt`：
```
https://site.com/chapter1
https://site.com/chapter2
https://site.com/chapter3
```

```bash
# 批量爬取并翻译
cat urls.txt | webmdai fetch pipe -n novel -st
for file in novel/*.md; do
    cat "$file" | webmdai llm pipe -t translate -o "translated_$(basename $file)"
done
```

### 案例3：内容清理流水线

```bash
# 多级处理：爬取 → 清理 → 翻译 → 保存
cat dirty_article.md | \
    webmdai deal pipe -f "ADVERTISEMENT" | \
    webmdai deal pipe --re -f "\n\n+" -r "\n\n" | \
    webmdai llm pipe -t translate -o final.md
```

## 平台差异说明

### Unix/Linux/macOS
```bash
# 结束管道输入：Ctrl+D
echo "content" | webmdai llm pipe -t translate
# 或直接在命令中使用
```

### Windows PowerShell
```powershell
# 结束管道输入：Ctrl+Z 然后 Enter
# 或从文件/变量传递
$content = Get-Content article.md -Raw
$content | webmdai llm pipe -t translate
```

### Windows CMD
```cmd
:: 结束管道输入：Ctrl+Z 然后 Enter
:: 或使用 echo
echo Hello | webmdai llm pipe -t translate
```

## 管道 vs 工作流选择指南

| 你的需求 | 推荐方式 | 原因 |
|---------|---------|------|
| 每天翻译最新章节 | 工作流 | 配置一次，重复使用 |
| 临时测试几个链接 | 管道 | 快速，无需配置文件 |
| 配合其他工具处理 | 管道 | 可以组合 grep/awk/sed 等 |
| 需要多阶段自动处理 | 工作流 | fetch→clean→translate→merge 一键完成 |
| 探索性处理 | 管道 | 即兴组合，灵活调整 |

## 高级技巧

### 1. 错误处理
```bash
# 如果失败不中断
echo "https://example.com" | webmdai fetch pipe -n test || echo "Fetch failed"

# 检查是否成功再执行下一步
if echo "https://example.com" | webmdai fetch pipe -n test; then
    cat test/*合并.md | webmdai llm pipe -t translate
fi
```

### 2. 批量并行处理（Linux）
```bash
# 使用 xargs 并行处理
cat urls.txt | xargs -P 4 -I {} sh -c 'echo {} | webmdai fetch pipe -n "task_$(date +%s)"'
```

### 3. 进度监控
```bash
# 使用 pv 显示进度（Linux）
cat large_file.md | pv | webmdai llm pipe -t summarize
```

## 常见问题

### Q: 管道输入如何结束？
A: 
- Unix/Linux/macOS: `Ctrl+D`
- Windows: `Ctrl+Z` 然后按 `Enter`

### Q: 为什么我的中文显示乱码？
A: 确保文件编码是UTF-8：
```bash
# PowerShell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

### Q: 管道可以处理多大内容？
A: 受限于内存，建议单文件不超过10MB。大文件建议分批处理。

### Q: 如何调试管道问题？
A: 添加 `-o` 参数保存中间结果：
```bash
cat input.md | webmdai deal pipe -f "test" -o debug.md
```
