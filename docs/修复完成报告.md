### 3. 管道（Pipe）支持 ✅

**新增命令**:

#### `fetch pipe` - 从管道读取URL
```bash
echo "https://example.com" | webmdai fetch pipe -n mytask
cat urls.txt | webmdai fetch pipe -n batch -st
```

#### `llm pipe` - 从管道读取内容处理
```bash
echo "Hello World" | webmdai llm pipe -t translate
cat article.md | webmdai llm pipe -t summarize -o summary.md
```

#### `deal pipe` - 从管道读取内容替换
```bash
echo "hello world" | webmdai deal pipe -f "world" -r "Python"
cat article.md | webmdai deal pipe -f "广告" -o cleaned.md
```

**修改文件**: `webmdai/cli.py`

**状态**: ✅ 已测试通过

## 📚 使用建议

### 新手推荐流程

1. **配置模型**（只需一次）
```bash
webmdai model add --name deepseek --endpoint https://openrouter.ai/api/v1 --model deepseek/deepseek-chat --key "你的密钥"
```

2. **简单任务用管道**（临时处理）
```bash
echo "https://example.com" | webmdai fetch pipe -n temp -t && \
cat temp/*合并.md | webmdai llm pipe -t translate -o result.md
```

3. **复杂任务用工作流**（重复执行）
```bash
# 创建 TASK.md 和 workflow.yaml
webmdai workflow run
```
