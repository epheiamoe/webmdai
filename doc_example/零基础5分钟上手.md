# WebMDAI 零基础5分钟上手

> 完全不懂编程？没关系！跟着这5步就能用起来。

## 🎯 一句话理解WebMDAI
**把网页内容变成你想要的格式**，比如：英文→中文、长文章→摘要、小说→翻译版。

## 📦 第一步：安装（1分钟）
```bash
# 1. 确保有Python 3.8+
python --version

# 2. 安装webmdai
cd /path/to/webmdai_clean  # 进入项目目录
pip install -e .
```

## 🔧 第二步：配置AI模型（1分钟）
```bash
# 添加一个AI模型（以OpenRouter为例）
webmdai model add \
  --name myai \
  --endpoint https://openrouter.ai/api/v1 \
  --model deepseek/deepseek-chat \
  --key your-api-key-here

# 设置为默认模型
webmdai model set-default myai
```

## 🚀 第三步：创建第一个项目（2分钟）

### 3.1 创建项目目录
```bash
# 随便找个地方，比如桌面
cd ~/Desktop
mkdir my_first_project
cd my_first_project
```

### 3.2 创建3个必要文件
```bash
# 文件1：任务文件（告诉webmdai要处理哪些网页）
cat > TASK.md << 'EOF'
# 我的测试任务

## 要处理的网页
- [示例页面](https://example.com)
EOF

# 文件2：提示词文件（告诉AI怎么处理）
mkdir -p prompts
cat > prompts/simple.txt << 'EOF'
请翻译以下内容为中文：{content}
EOF

# 文件3：工作流配置（告诉webmdai整个流程）
cat > workflow.yaml << 'EOF'
name: 简单翻译测试
stages:
  - name: 爬取网页
    type: fetch
    params:
      source: taskfile
      taskfile: TASK.md
  
  - name: 翻译
    type: llm
    params:
      model: default
      prompt_file: prompts/simple.txt
      file_pattern: "*.md"
EOF
```

## ▶️ 第四步：运行！（1分钟）
```bash
# 在项目目录中运行
webmdai workflow run workflow.yaml
```

## 📁 第五步：查看结果

### 下一步做什么？
1. **修改TASK.md**：换成你真正想处理的网页链接
2. **修改prompts/simple.txt**：调整提示词，让AI按你的要求处理
3. **修改workflow.yaml**：添加更多处理步骤

### 常用修改示例

#### 修改1：处理多个网页
```markdown
# TASK.md 修改后
# 我的翻译任务

## 技术文章
- [Python教程](https://docs.python.org/3/tutorial/)
- [GitHub](https://github.com)
- [Stack Overflow](https://stackoverflow.com)
```

#### 修改2：改进提示词
```txt
# prompts/better.txt
请将以下英文技术文档翻译成简体中文，要求：
1. 专业术语准确
2. 语句通顺
3. 保留代码示例
4. 技术名词首次出现时加注英文

内容：{content}
```

#### 修改3：添加清理步骤
```yaml
# workflow.yaml 修改后
name: 改进的工作流
stages:
  - name: 爬取
    type: fetch
    params:
      source: taskfile
      taskfile: TASK.md
  
  - name: 清理内容
    type: clean
    params:
      file_pattern: "*.md"
      preset: "technical_docs"
  
  - name: 翻译
    type: llm
    params:
      model: default
      prompt_file: prompts/better.txt
      file_pattern: "*_cleaned_*.md"
```

## ❓ 常见问题

### Q：文件应该放在哪里？
**A：全部放在同一个文件夹里！**
- 进入你的项目文件夹：`cd my_first_project`
- 所有文件都创建在这里
- 运行命令也在这里

### Q：提示词里的`{content}`是什么？
**A：这是占位符**，webmdai会自动把网页内容放进去。

### Q：运行出错怎么办？
检查这3点：
1. ✅ 是否在项目目录里？（运行`pwd`查看）
2. ✅ 3个文件都存在吗？（运行`ls -la`查看）
3. ✅ AI模型配置好了吗？（运行`webmdai model list`查看）

### Q：想处理自己的网页？
1. 打开TASK.md
2. 把你的网页链接加进去
3. 保存，重新运行
